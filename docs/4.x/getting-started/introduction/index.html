Introduction

Auto Clicker – Auto Fill helps you automate clicks and form fills on any website by defining targets and actions.

On this page
  • Quick start
  • Additional (Advance)
  • New to the extension?
    • Try now
    • Set up Global Settings (recommended)
  • Community

Quick start

  1. Download the extension and install it in Chrome or Edge.
  2. Open the web page where you want to automate clicks or fills.
  3. Right‑click any element (button, input, textarea, etc.) and choose Auto Clicker - AutoFill.
  • Pick ★ Configure for this Field to auto-fill:
    • The page URL
    • The Element Finder for the selected element
  • Or pick ☉ Auto Clicker (Record) to record your actions and auto‑create a configuration.

Additional (Advance)

  • Configuration List - it consist of all configuration
    • Configuration - combination of url and actions
      • Configuration Settings
      • Batch - set of actions
        • Action - combination of element finder and action which need to be performed.
        • Action Value
        • Action Settings
        • Action Condition
          • Addon - add condition before action
  • Global Settings - global setting how extension should work

Try now

  • URL: https://test.getautoclicker.com/
  • Goal: Fill username and click a button

Steps

  1. Open the page above.
  2. Add Action 1
  • Element Finder: //input[@name="username"]
  • Value: Your name
  1. Add Action 2
  • Element Finder: //button[@random]
  • Value: leave blank (blank value triggers click)
  1. Save and run.
  • Use right‑click → Auto Clicker - AutoFill → ★ Configure for this Field to auto‑capture the Element Finder.
  • If the page loads slowly, set an Init Wait.

Set up Global Settings (recommended)

Global settings control how the extension behaves across all configurations.

  1. Open Global Settings.
  2. Consider setting:
  • Start mode: Auto or Manual
  • Hotkey: Define a shortcut to start in Manual mode (e.g., Ctrl + Shift + A)
  • Extension Load event: Window vs Document for faster startup
  • Status Bar location: Adjust or disable if needed
  1. Save changes and refresh your target page.
