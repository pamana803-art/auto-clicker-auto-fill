## Supported Patterns

<BsTable>

| **Pattern**          | **Description**                                                                                                             | **Example**                                                     |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| Query::param         | Replace with the value of query parameter param from the URL. If not found, returns the key.                                | Query::userId → 123 (if ?userId=123 in URL)                     |
| &lt;query::param&gt; | Replace all occurrences of &lt;query::param&gt; with the value of query parameter param from the URL. Useful for templates. | Hello &lt;query::name&gt;! → Hello John! (if ?name=John in URL) |

</BsTable>

### Usage

Use <code>Query::param</code> for single value replacement, and <code>&lt;query::param&gt;</code> for replacing within a string or template.
