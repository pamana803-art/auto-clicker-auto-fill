<Callout type="info">
  This is the target URL where the extension executes actions. You may use a plain URL or a [regular
  expression](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions) to match multiple
  pages.
</Callout>

### Examples

##### https://www.instagram.com/dhruv.techapps/

- `https://www.instagram.com/dhruv.techapps/`
- `https://www.instagram.com/dhruv`
- `https://www.instagram.com/`
- `https://.*.instagram.com/`
- `.*://.*.instagram.com/`
